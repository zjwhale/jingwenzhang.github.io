# Personal Website

A simple, GitHub Pages-friendly personal website with no build step.

## What's inside

- `index.html`: your content and page structure
- `styles.css`: the design system and responsive layout
- `script.js`: small enhancements for section reveals and footer year
- `.nojekyll`: tells GitHub Pages to serve the site as plain static files

## Customize it

Open `index.html` and replace the bracketed placeholder text:

- `[Your Name]`
- `[Researcher, developer, designer, or your role]`
- `[City, Country]`
- project titles, links, and contact details

You can also tweak colors in `styles.css` by editing the variables near the top of the file.

## Publish on GitHub Pages

### Option 1: main personal site

1. Create a new repository named `<your-github-username>.github.io`
2. Upload the contents of this folder to that repository
3. Push to the `main` branch
4. In GitHub, go to `Settings -> Pages`
5. Set the source to `Deploy from a branch`
6. Select branch `main` and folder `/ (root)`

Your site should appear at:

`https://<your-github-username>.github.io`

### Option 2: project site

1. Create any repository name, for example `personal-website`
2. Upload the contents of this folder
3. Enable GitHub Pages from `main` and `/ (root)`

Your site should appear at:

`https://<your-github-username>.github.io/<repo-name>/`

## Preview locally

From this folder, run:

```bash
python3 -m http.server 8000
```

Then open:

`http://localhost:8000`
